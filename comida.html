BIENVENIDOS A

SCAP

El proyecto consiste en desarrollar un sistema de comedero automatizado para perros, con una página web-aplicación web accesible desde cualquier dispositivo con conexión a internet. Los usuarios podrán programar horarios de alimentación específicos para sus mascotas y el sistema activará una luz LED en el comedero para captar la atención de los perros. Además, la página web incluirá un apartado informativo sobre 5 tipos de alimentación de perros. Este sistema será útil para dueños de mascotas, criadores, servicios de cuidado de mascotas y tiendas de mascotas que deseen tener un método automatizado y programable de alimentación para perros.
